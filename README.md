# FullStackDiaryThoughtsJournalApplication

What My appliication does:

A diary/daily notes entry tracker. You can log in and write daily notes/entries and attach tags to each one and be able to sort or filter entries by the tags and also see a short summary of each entry. Target users: anyone who wants to keep track of daily thoughts and look back at similar entries.

Features
- Create, read, update, and delete Tags and Entries
- Backend API with full CRUD operations
- Data persisted in MongoDB
- Advanced feature:
    - filter and sort by entries by the tags they contain
- Error handling on client + server

Advanced Features
- This application uses proper encryption
- It creates cookies and also uses json web tokens (JWT) to ensure only the individual user's data is being shown and can be interacted with
